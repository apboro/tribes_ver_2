## класс App\Services\Telegram\TelegramApi\Mtproto для взаимодействия с телеграмом 
```php
методы: 
"auth" // атворизация пользователя. Первый запрос на авторизацию должен содержать только id и номер телефона пользователя у которого уже есть аккаунт в телеграме иначе будет ошибка. После запроса на телеграм аккаунт пользователя придет сообщение с кодом доступа. Инициализируем второй запрос на авторизацию с кодом доступа, в случае успеха вернёт информацию о пользователе. 
"logOut" // Завершение открытого сеанса.
'getDialogs' // Получить диалоги пользователя и информацию о них. Необходимо делать для получения access_hash
'getMessages' // получение сообщений из телеграма. Содержит полную информацию о сообщениях. (реакции, просмотры, репосты)
'getChatInfo' // Возвращает информация о ГРУППЕ в том числе и всех её участников
'getUsersInChannel' // Получить информацию о пользователях КАНАЛА
